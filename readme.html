This is the readme for the models associated with the paper:

Eyal G, Verhoog MB, Testa-Silva G, Deitcher Y, Benavides-Piccione R, DeFelipe J, de Kock CPJ, Mansvelder HD, Segev I (2018) Human cortical pyramidal neurons: From spines to spikes via models Frontiers
https://www.frontiersin.org/articles/10.3389/fncel.2018.00181

This model was contributed by Guy Eyal. Please download and extract the (133MB) file from this link: http://senselab.med.yale.edu/modeldb/data/238347/EyalEtAl2018.zip

Consult the zip files in the figure subfolders to run the simulations.
